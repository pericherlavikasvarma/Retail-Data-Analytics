# Customer Segmentation Analysis
Using Python to perform RFM (Recency, Frequency, Monetary) analysis on retail data.

## Key Features
* **Data Cleaning:** Handled missing values and returns.
* **Segmentation:** Grouped customers into Champions, Loyal, and At-Risk.
* **Visualization:** Includes 3D interactive Plotly clusters and Snake Plots.

## Tools Used
* Python (Pandas, Seaborn, Plotly)
* Jupyter Notebooks
